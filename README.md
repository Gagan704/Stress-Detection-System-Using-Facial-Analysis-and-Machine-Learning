# Stress Detection in Professionals Using Mobile-Net-based CNN and Multi-Modal Analysis

## Overview

This project presents an AI-powered multimodal stress detection system that predicts stress levels using facial expression analysis and physiological information. The system combines MobileNet-based Convolutional Neural Networks (CNNs) with computer vision techniques to improve stress detection accuracy.

The project was developed as a final-year Bachelor of Engineering project at Atria Institute of Technology, Bengaluru.
